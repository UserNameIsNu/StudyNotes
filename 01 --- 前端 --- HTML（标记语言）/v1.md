# HTML

# 1，简介与概念

## 1.这是啥

​	HTML（Hypertext Markup Language）超文本标记语言。

​	它通过一系列的标签（元素）来定义文本、图像、链接等等。

​	一般就是尖括号包起来的关键字。

## 2.了解单双标签

​	标签通常成对出现，包括开始标签和结束标签（也称为双标签）。

​	内容位于这两个标签之间，例如:

```html
<p>这是一个段落。</p>
<h1>这是一个标题。</h1>
<a href="#">这是一个超链接。</a>
<!-- ... -->
```

​	孤家寡人的也有，例如：

```html
<input type="text">
<br>
<hr>
<!-- ... -->
```

## 3.熟悉注释

​	HTML 中的注释长下面这样：

```html
<!-- 注释内容 -->
```

​	注释是不会显示的，不管写啥都不会在页面中看到。

​	只有打开 HTML 文档查看代码才能看到。

## 4.熟悉基本结构

​	HTML 文档起码要有这些东西。

```html
<!-- 告知浏览器这是HTML文件 -->
<!DOCTYPE html>
<!-- 文档的开始与结束范围 -->
<html>
    <!-- 这里放置文档的元信息 -->
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
        <!-- ... -->
    </head>
    <!-- 这里放置页面内容（页面中能看见的大多就在这里了） -->
    <body>
        <!-- ... -->
    </body>
</html>
```

# 2，常用标签

## 1.结构标签（div）

​	en...

​	收纳箱？

​	把各种标签包起来，然后摆放。

​	东西进了箱子，那不就可以整箱整箱的动位置了嘛。

​	箱子当然也能装其它箱子嘛。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <div>
        	<h1>一级标题</h1>
            <h2>二级标题</h2>
            <h3>三级标题</h3>
            <div>
                <h4>四级标题</h4>
                <h5>五级标题</h5>
                <h6>六级标题</h6>
            </div>
        </div>
    </body>
</html>
```

## 2.文本标签（h1-6，p，span，br，hr，b，i，u，s）

​	最常用的也就是下面这些了。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 标题标签 -->
        <h1>一级标题</h1>
        <h2>二级标题</h2>
        <h3>三级标题</h3>
        <h4>四级标题</h4>
        <h5>五级标题</h5>
        <h6>六级标题</h6>
        <!-- 段落与选段标签 -->
        <p>文<span>本</span></p>
        <!-- 换行标签 -->
        <br>
        <!-- 水平线标签 -->
        <hr>
        <p><b>加粗</b>，<i>斜体</i>，<u>下划线</u>，<s>删除线</s></p>
    </body>
</html>
```

## 3.列表标签（ul，ol，li）

​	无序和有序列表。

​	它们俩的区别也就是一个列表头是图形，一个列表头是数字字母啥的罢了。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 无序列表标签 -->
        <ul>
            <!-- 列表项标签 -->
            <li>1</li>
            <li>2</li>
            <li>3</li>
        </ul>
        <!-- 有序列表标签 -->
        <ol>
            <li>1</li>
            <li>2</li>
            <li>3</li>
        </ol>
    </body>
</html>
```

## 4.表格标签（table，tr，th，td）

​	基本类似 excel 的表格（当然你可以让它变得不一样）。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 表格标签 -->
        <table border="1px">
            <!-- 行 -->
            <tr>
                <!-- 标题格标签 -->
                <th>A</th>
                <th>B</th>
            </tr>
            <tr>
                <!-- 常规格标签 -->
                <td>A1</td>
                <td>B1</td>
            </tr>
            <tr>
                <td>A2</td>
                <td>B2</td>
            </tr>
        </table>
    </body>
</html>
```

## 5.媒体标签（img，video，audio，source）

​	图片，视频，音频与媒体源。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 图片标签 -->
        <img src="example.jpg">
        <!-- 视频标签 -->
        <video src="video.mp4"></video>
        <!-- 音频标签 -->
        <audio src="audio.mp3"></audio>
        <!-- 媒体源标签（如果出现了不支持的格式，就会试试下一种） -->
        <video controls>
  			<source src="movie.mp4" type="video/mp4">
  			<source src="movie.ogg" type="video/ogg">
  			您的浏览器不支持视频标签。
		</video>
    </body>
</html>
```

## 6.超链接标签（a）

​	跳转用的。

​	能跳页面，也能跳图片和指定位置。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 页面跳转 -->
        <a href="https://www.example.com">访问示例网站</a>
        <!-- 锚点跳转（去到指定id属性值的地方） -->
        <a href="#idName">跳转到第一部分</a>
        <!-- 外部锚点跳转（去到其它页面指定id属性值的地方） -->
        <a href="https://www.example.com#idName">访问示例网站</a>
        <!-- 点击图片跳转 -->
        <a href="https://www.example.com">
          <img src="logo.png">
        </a>
    </body>
</html>
```

## 7.表单标签（form，label，input）

​	就像给别人一张问卷，人家写完后提交，你就能看到人家提交的问卷内容了。

​	当然这个问卷的内容是自己定的。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <!-- 表单标签 -->
        <form action="...">
            <!-- 表单项注释标签 -->
            <label>用户名：</label>
            <!-- 表单项标签 -->
            <input type="text" name="username" placeholder="请输入用户名">
            <input type="password" name="password" placeholder="请输入密码">
            <input type="radio" name="gender" value="male"> 男
			<input type="radio" name="gender" value="female"> 女
            <input type="checkbox" name="hobby" value="reading"> 阅读
			<input type="checkbox" name="hobby" value="sports"> 运动
            <input type="file" name="file">
            <input type="date" name="birthday">
            <input type="reset" value="重置">
            <input type="submit" value="提交">
        </form>
    </body>
</html>
```

# 3，标签属性

## 1.干嘛的

​	用于定义标签的行为，外观或与其它标签的关系。

## 2.数量问题

​	标签可以同时拥有多个属性。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    <body>
		<p id="describe" class="section">这是一个段落标签</p>
        <a href="https://www.baidu.com">跳转至百度</a>
    </body>
</html>
```

## 3.大小写问题

​	属性名不区分大小写，但是属性值要区分。

​	下面前两个可以跳转，第三个不行。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    	<!-- 假设有一张叫‘example.jpg’的图片 -->
		<img src="example.jpg">
        <img SRC="example.jpg">
        <img src="EXAMPLE.JPG">
    </body>
</html>
```

## 4.常用的通用属性（id，class，style，width，height）

​	也不是全都能用，但也算是几乎都能用了（超过一个就不算都有了不是吗）。

​	主要就下面几个。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
    	<!-- 定义唯一标识 -->
		<p id="a">这是一个段落标签</p>
    	<!-- 定义组标识 -->
    	<p class="b">这是一个段落标签</p>
    	<!-- 定义行内样式（CSS内容） -->
    	<p style="...">这是一个段落标签</p>
    	<!-- 定义标签宽度 -->
    	<img src="example.jpg" width="100px">
    	<!-- 定义标签高度 -->
    	<img src="example.jpg" height="100px">
    	<!-- 定义隐藏标签 -->
    	<img src="example.jpg" hidden>
    </body>
</html>
```

## 5.常用的独有属性（target，download，alt，action，type，placeholder，required，name，value）

​	除了通用属性，有些标签还有独有属性，可以发挥一些特别的作用。

​	这里不细说，就提一嘴，详细属性值自己搜。

​	`input` 标签属性值是真多。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
    </head>
		<!-- ‘target’定义链接的打开方式 -->
    	<a href="https://www.baidu.com" target="...">跳转至百度</a>
    	<!-- ‘download’定义链接不打开而是下载 -->
    	<a href="https://www.baidu.com" download>跳转至百度</a>
    	<!-- ‘alt’定义图片不显示时的提示文本 -->
    	<img src="example.jpg" alt="图片不存在">
    	<!-- ‘action’定义表单发送的目标地址 -->
    	<form action="...">
            <!-- ‘type’定义表单项标签的形式（文本框，密码框，单选框，多选框，重置，提交...） -->
            <input type="...">
            <!-- ‘placeholder’定义输入框提示语 -->
            <input type="text" placeholder="...">
            <!-- ‘required’定义内容必填 -->
            <input type="text" required>
            <!-- ‘value’定义输入框标识名与值 -->
            <input type="text" name="..." value="...">
    	</form>
    </body>
</html>
```

# 4，块级与行级标签

## 1.块级标签

​	块级元素通常用于组织和布局页面的主要结构和内容，例如段落、标题、列表、表格等，它们用于创建页面的主要部分。

​	块级元素通常会从新行开始，并占据整行的宽度。

​	可以包含其他块级元素和行级元素。

​	常见的块级标签有：结构标签，段落标签，标题标签，有无序列表标签，列表项标签，表格标签，表单标签等。

## 2.行级标签

​	行级元素通常用于添加文本样式或为文本中的一部分应用样式，它们可以在文本中插入小的元素，例如超链接强调文本等。

​	行级元素通常在同一行内呈现，不会独占一行。

​	它们只占据其内容所需的宽度，而不是整行的宽度。

​	行级元素不能包含块级元素，但可以包含其他行级元素。

​	常见的行级元素有：选段标签，超链接标签，文本样式标签，换行标签等。
