# CSS

# 1，简介与概念

## 1.这是啥

​	CSS（cascading style sheets）层叠样式表。

​	用于定义网页样式和布局的样式表语言。

​	通过 CSS，你可以指定页面中各个元素的颜色、字体、大小、间距、边框、背景等样式，从而实现更精确的页面设计。

​	HTML 就是毛坯房，这玩意就相当于开始精装修。

## 2.了解语法与注释

​	一般就是选择器 + 属性 + 属性值组成。

​	可以同时应用多条属性。

​	每条属性独占一行，并用分号结尾。

```css
p {
    /* 这是注释 */
	color: blue;
	font-size: 16px;
}
```

## 3.了解使用（外部，内部，行内）

​	常用的就是外部样式，内部样式与行内样式三种。

​	外部样式就是在 HTML 文档中引入 CSS 文件。

​	内部与行内样式是直接在 HTML 文件中写。

​	三种方式是有应用优先级的，行内 > 内部 > 外部。

```css
p {
    /* 外部样式 */
	color: blue;
	font-size: 16px;
}
```

```html
<!DOCTYPE html>
<html>
    <head>
        <title>文档标题</title>
        <meta charset="UTF-8">
        <style>
            p {
                /* 内部样式 */
                color: blue;
                font-size: 16px;
            }
        </style>
    </head>
    <body>
        <!-- 行内样式 -->
        <p style="color: red;">123</p>
    </body>
</html>
```

# 2，选择器

## 1.干嘛的

​	这玩意是 CSS 的核心部分。

​	一句话。

​	可以允许通过特定规则来选取目标标签从而定义样式还不影响其它标签。

## 2.八种选择器（标签选择器，类选择器，ID选择器，通配选择器，子元素选择器，后代选择器，兄弟选择器，伪类选择器）

​	标签选择器。	

​	就是直接只用标签名，选中文档中的所有这个标签。

```CSS
/* 选中文档中所有的段落标签 */
p {
	color: blue;
	font-size: 16px;
}
```

​	类选择器。

​	回忆一下 HTML 中的通用属性中的 `class` 。

​	这玩意就是用于选中指定的 `class` 名的标签。

```css
/* 选中文档中所有的类名为这玩意的标签 */
.className {
	color: blue;
	font-size: 16px;
}
```

​	ID选择器。

​	也是和 HTML 中的通用属性相关。

​	这个指定的是 `id` 名。

```css
/* 选中文档中所有的ID名为这玩意的标签 */
#idName {
	color: blue;
	font-size: 16px;
}
```

​	通配选择器。

​	选中文档中的所有标签元素。

​	（就直接开地图炮了）。

```css
/* 选中文档中所有的标签 */
* {
	color: blue;
	font-size: 16px;
}
```

​	子元素选择器。

​	选中某的元素内的某个（某些）元素。

​	这个内部元素也是其它选择器选出来的。

​	不过要注意它的名字，**子**选择器，当然只能选中某个元素内的直属一级的元素。

​	比如还有个孙子元素就选不到了。

```css
/* 选中文档中所有的 div 标签内的直属一级的所有的这个类名的标签 */
div > .className {
	color: blue;
	font-size: 16px;
}
```

​	后代选择器。

​	上面的只能选儿子，这个可以选全家（比自己小的全家）,那么当然连着啥孙子曾孙子都能选中啦。

```css
/* 选中文档中所有的 div 标签内的所有的这个类名的标签 */
div .className {
	color: blue;
	font-size: 16px;
}
```

​	兄弟选择器。

​	选中同级的一个好兄弟。

​	不过只能是**紧跟着**的，**后面**的，**一个**，好兄弟。

```css
/* 选中文档中所有的这个id名的标签后面的一个 p 标签 */
#idName + p {
	color: blue;
	font-size: 16px;
}
```

​	伪类选择器。

​	不知道为啥这么叫，反正这个用于选中鼠标相关操作（一部分操作，比如悬停），链接状态或第某个元素的标签。

​	一般是 `:` 开头的似乎就算是伪类选择器了。

```css
/* 鼠标悬停在 p 标签上时应用这个样式 */
p:hover {
	color: blue;
	font-size: 16px;
}
```

# 3，常用样式

## 1.字体文本与颜色样式（font-...，text-...，color，background-color）

​	下面的样式不是全部，只是部分（毕竟是极速版文档），认识一下先。

​	首先，当然是字体这个最简单直接的小玩意。

​	先是字体基本样式。

```css
p {
    font-family: Arial, sans-serif; /* 设置字体 */
    font-size: 16px; /* 设置字体大小 */
    font-weight: bold; /* 设置字体粗细，如 bold、normal */
    font-style: italic; /* 设置字体风格，如 italic */
}
```

​	然后是文本对齐与装饰样式。

```css
p {
    text-align: center; /* 文本居中 */
    text-align: left; /* 文本左对齐 */
    text-align: right; /* 文本右对齐 */
    text-decoration: underline; /* 下划线 */
    text-decoration: line-through; /* 删除线 */
    text-decoration: none; /* 取消链接的默认下划线 */
    text-shadow: 2px 2px 4px #000; /* 水平偏移，垂直偏移，模糊半径，颜色 */
}
```

​	最后就是颜色，分为字体颜色与背景颜色。

```css
div {
    color: #ff0000; /* 设置文本颜色 */
	background-color: #f0f0f0; /* 设置背景颜色 */
}
```

## 2.宽高与边距边框样式（width，height，padding，margin，border-...）

​	先是宽高两个基本又简单理解的样式。

```css
div {
    width: 200px; /* 设置宽度 */
	height: 100px; /* 设置高度 */
}
```

​	然后是内外边距。

​	边距和边框样式与下面第四章有关系，可以扫一下，第四章结束了回来看就知道是个啥了。

```css
p {
    padding: 10px; /* 上下左右内边距均为10px */
    padding: 10px 20px; /* 上下10px，左右20px */
    padding: 10px 20px 30px 40px; /* 上10px，右20px，下30px，左40px */
    margin: 10px; /* 上下左右外边距均为10px */
    margin: 10px 20px; /* 上下10px，左右20px */
    margin: 10px 20px 30px 40px; /* 上10px，右20px，下30px，左40px */
}
```

​	最后就是边框了。

```css
p {
    border: 1px solid #000; /* 1px 宽度，实线，黑色 */
	border-radius: 10px; /* 圆角边框 */
}
```

## 3.浮动与定位样式（float，clear，position，top，left，right，bottom）

​	这两个属于布局，也是第四章的内容，同样扫一下即可，看完第四章再回来就懂了（不过你只是把这个文档当字典来用那就没关系了）。

​	先是浮动。

```css
p {
    float: left; /* 向左浮动 */
    float: right; /* 向右浮动 */
    clear: both; /* 清除浮动 */
}
```

​	然后就是定位了。

​	记得不要给同一个元素不同的定位方式，很大几率会炸掉（位置乱飞，自己都不知道这玩意怎么去到那里的）。

​	不过比浮动好用（浮动飞的更吓人）。

```css
p {
    position: static; /* 静态定位（默认值） */
}
p {
    position: relative; /* 相对于自身位置进行偏移 */
    top: 10px;
    left: 20px;
}
p {
    position: absolute; /* 相对于最近的定位父元素进行偏移 */
    bottom: 10px;
    left: 20px;
}
p {
    position: fixed; /* 相对于浏览器窗口进行偏移 */
    top: 10px;
    right: 20px;
}
p {
    position: sticky; /* 在滚动到指定位置时变为固定定位 */
	top: 0;
}
```

## 4.弹性与网格布局样式（display，flex-...，justify-...，align-...，flex，order，grid-...，gap）

​	还是第四章的内容。

​	就是 flexbox 和 grid 两种布局的容器定义与子项设置用的。

​	先是 flexbox ，也就是弹性布局样式。

```css
/* 弹性布局容器 */
div {
    display: flex; /* 启用弹性布局 */
    flex-direction: row; /* 主轴方向：水平（默认） */
    flex-direction: column; /* 主轴方向：垂直 */
    justify-content: center; /* 主轴对齐方式：居中 */
    align-items: center; /* 交叉轴对齐方式：居中 */
}
/* 弹性布局子项 */
.son {
    flex: 1; /* 子项占据剩余空间 */
	order: 2; /* 子项的排序顺序 */
}
```

​	然后就是 grid 网格布局样式。

```css
/* 网格布局容器 */
div {
    display: grid; /* 启用网格布局 */
    grid-template-columns: 100px 200px auto; /* 定义列的宽度 */
    grid-template-rows: 50px 100px; /* 定义行的高度 */
    gap: 10px; /* 网格之间的间隔 */
}
/* 网格布局子项 */
.son {
    grid-column: 1 / 3; /* 子项跨越的列 */
	grid-row: 1 / 2; /* 子项跨越的行 */
}
```

## 5.背景样式（background-...）

​	背景颜色上面提过了嘛，这里就加个背景图片吧。

```css
div {
    background-image: url('image.jpg');
    background-repeat: no-repeat; /* 背景图片不重复 */
    background-size: cover; /* 背景图片覆盖整个元素 */
    background-position: center; /* 背景图片居中 */
}
```

## 6.阴影动画与响应式样式（box-...，@keyframes，from，to，animation，transition，@media）

​	最后的样式就是阴影动画与响应式了。

​	先来阴影，也分文本阴影与背景阴影。

​	文本阴影上面有了就不重复了。

```css
div {
    box-shadow: 2px 2px 4px #000; /* 水平偏移，垂直偏移，模糊半径，颜色 */
}
```

​	然后是动画。

​	动画也分主动动画与过度动画。

​	主动动画就是自己动嘛。

​	过渡动画就是当这个元素发生如位置或大小等变化时，沿着移动路径或膨胀收缩范围等自动生成的相当于填充帧（你看懂了吗？我是看懂了）。

​	不过动画可没这么简单，这里只是简单提一嘴罢了（极速版特色）。

```css
/* 主动动画 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
div {
    animation: fadeIn 1s ease-in-out;
}
/* 过渡动画 */
div {
    transition: all 0.3s ease; /* 所有属性，持续时间，过渡效果 */
}
```

​	最后就是响应式样式了。

​	严格来说也不算样式，毕竟它本身不定义什么实际样式，它就是指定页面在某个大小时应用不同的样式组。

```css
@media (max-width: 1920px) {
  body {
    font-size: 12px;
  }
}
@media (max-width: 720px) {
  body {
    font-size: 15px;
  }
}
```

# 4，盒子模型与布局

## 1.啥玩意

​	盒子模型就是一种规范。

​	规定了一个元素占用的位置。

​	而布局主要就是一种方便页面构建的方式。

## 2.分层（内容，内边距，边框，外边距）

​	有四层盒子。

​	最里面的是内容层（content），就是这个元素本身。

​	然后是内边距（padding），就是内容与边框的距离，这玩意只是间距，看不到的。

​	然后是边框（border），能看到的，就像描了个边。

​	最后是外边距（margin），就是整个元素盒子（所有层在一起）与其它元素盒子的距离，也看不到的。

## 3.五种布局（标准流，浮动，定位，flexbox，grid）

​	只是按照不同的规则摆盒子。

- 标准流就是按照行级块级等这些默认的规定，从左往右，从上到下，逐一排序元素。

​	下面四个都不给的话，默认就是这个。

- 浮动，字面意思，看过平静水面上的叶子吗？一般都是向着边上漂是吧，这个就是吧元素脱离标准流，向左或右漂到边上。

​	注意只会在所在父元素里面漂（叶子在自己的盆子里漂，当然可以把这个装着叶子的盆子放进更大的盆子继续漂）。

​	如果一行内浮动的叶子太多了，剩余的会被挤到下一行（不会掉出盆子消失走丢啥的）。

​	再有了，浮动之后这个元素的宽高将不会影响父元素了，这个浮动元素再怎么大也不会把父元素撑起来。

- 定位，直接把元素强行按在某个地方，常用的就三种定位。

  相对定位就是相对于这个元素的当前位置开始动。

  绝对定位就是相对于这个元素的父元素（没有父元素就相对到页面窗口）的位置开始定。

  固定定位就是相对于页面窗口开始定。

  绝对和固定定位似乎都是以左上角为起点。

- flexbox 弹性布局，可以让元素宽高位置等根据需要自动调整适应。

​	一般就是给父元素设置为弹性容器，然后内部的子元素就可以开始应用弹性布局了。

- grid 网格布局，相当于把元素全部变成表格，然后去调整表格的行列格。

​	也是将父元素设置为网格布局容器后就可以对内部子元素应用了。
